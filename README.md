# Botãozinho JS

Projeto educacional interativo para aprender JavaScript de forma leve, prática e visual.

## ✨ Sobre o projeto

O Botãozinho JS é uma plataforma de estudos com trilhas, aulas, quiz, desafios práticos e preview visual.  
A proposta é ajudar iniciantes a entender JavaScript sem desespero, com exemplos simples, linguagem acessível e uma interface amigável.

## 🚀 Tecnologias usadas

- React
- Vite
- JavaScript
- CSS
- HTML

## 🧭 Funcionalidades

- Trilhas de aprendizado
- Aulas organizadas por níveis
- Quiz interativo
- Área de código
- Preview visual
- Tema claro e escuro
- Cards de progresso, usuário e conquistas

## 📚 Níveis

- Bebê JS
- Dev de Rodinhas
- Domador de Arrays
- Sobreviveu ao console.log

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
```
